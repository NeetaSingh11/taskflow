# 🚀 TaskFlow

A full-stack Task Management Application built using the MERN Stack with secure JWT authentication, task management, dashboard analytics, search, filtering, pagination, and a modern responsive UI.

---

## ✨ Features

- 🔐 User Registration & Login
- 🔑 JWT Authentication
- 🛡 Protected Routes
- ✅ Create, Update & Delete Tasks
- 📊 Dashboard Statistics
- 🔍 Search Tasks
- 🎯 Filter by Status & Priority
- 📄 Pagination
- 🔔 Toast Notifications
- 📱 Responsive Design
- ☁ MongoDB Atlas Integration

---

## 🛠 Tech Stack

### Frontend
- React.js
- React Router
- Axios
- React Toastify
- CSS

### Backend
- Node.js
- Express.js
- MongoDB
- Mongoose
- JWT
- bcrypt

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/NeetaSingh11/taskflow.git
```

### Backend

```bash
cd backend
npm install
npm run dev
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```
